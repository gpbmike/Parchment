Parchment An extension of WYSIHTML5

Requires jQuery, wysihtml5 and mustache.js.

<textarea
    data-parchment-commands='["fontstyles", "emphasis", "lists", "blockquote", "link", "image", "html"]'>
</textarea>
$('textarea').parchment();

So what?

The goal of this project is to easily customize rich text editors between apps.

Commands

A command is a plugin for the rich text editor. bold, italic and underline are all commands.

command, parserRules, init
